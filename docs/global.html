AuditOptionsObject

The options for fetching audit logs
Properties:
Name Type Argument Default Description
target String <optional>
 null The target's id (the user that the action was performed on)
actionType Number <optional>
 null The action type {@see https://discordapp.com/developers/docs/resources/audit-log#DOCS_AUDIT_LOG/audit-log-entry-object-audit-log-events}
before String <optional>
 null Filter the log before a certain entry ID
limit Number <optional>
 null How many entries are returned (default 50, minimum 1, maximum 100)

ChannelOptionsObject

The options for making/editing a channel
Properties:
Name Type Argument Default Description
name String <optional>
 null The name of the channel
position Number <optional>
 null Position of the channel
topic String <optional>
 null The topic of the channel (textchannel only)
nsfw Boolean <optional>
 null If the channel is NSFW or not (textchannel only)
bitrate Number <optional>
 null The bitrate of the channel (voicechannel only)
userlimit Number <optional>
 null The user limit of the channel (voicechannel only)
parent String <optional>
 null The parent channel's ID of the channel
